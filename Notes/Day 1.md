Day 1 - Arch Linux Installation
Date: June 9, 2026
Objective
Install Arch Linux on my ASUS TUF F15 and begin learning Linux through hands-on experience.
What I Learned
Today was my first serious attempt at installing Arch Linux. I followed installation guides and learned about bootable media, partitions, mounting drives, and the overall Linux installation process.
One of the biggest lessons from today was learning how important it is to verify which drive is being modified before running commands.
Challenges
During my first installation attempt, I accidentally installed Arch Linux onto my USB flash drive instead of the laptop's internal SSD. The installation itself completed successfully, but the operating system was located on the wrong device.
After discovering the mistake, I restarted the process from the beginning and carefully reviewed the drive layout before continuing.
This cost time, but it helped me better understand how Linux identifies storage devices and how easy it is to make mistakes when working directly from the command line.
Successes
Created a bootable Arch Linux installer
Booted into the Arch installation environment
Learned basic partitioning concepts
Learned how Linux identifies storage devices
Successfully completed a full Arch Linux installation
Recovered from an installation mistake without giving up
Key Takeaway
The installation was not perfect, but I successfully worked through mistakes and completed my first Arch Linux installation. Troubleshooting and correcting errors was one of the most valuable parts of the experience.
