Day 4
Date: June 13, 2026
Today I installed and experimented with Wireshark and Nmap.
Using Linux networking tools, I was able to identify:
My local IP address,
My default gateway,
My subnet,
The Docker network created on my system, and
Active services running on my laptop.
Using Nmap, I discovered that Open WebUI was running and listening on TCP port 8080. This helped me better understand how services communicate over a network and how network scanning can be used to identify running applications.
I also performed a network discovery scan of my local network and identified multiple active devices. While I was not able to definitively identify every device, I was able to determine which devices were online and make educated guesses about what some of them might be based on their network behavior.
